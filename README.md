# Jitendra Kumar Portfolio

A modern, responsive portfolio built with React, Vite, and Tailwind CSS.

## Features

- Projects showcase
- Blog
- Testimonials
- Downloadable CV (PDF)
- Contact form (EmailJS)
- Responsive design

## Getting Started

1. Clone this repo
2. Install dependencies:
   ```
   npm install
   ```
3. Start dev server:
   ```
   npm run dev
   ```
4. Build for production:
   ```
   npm run build
   ```
5. Deploy to GitHub Pages.
   ```
   npm run deploy
   ```

## Tailwind Setup

- See `tailwind.config.js` and `src/index.css` for Tailwind config.

## EmailJS Setup

- Create a free EmailJS account [https://www.emailjs.com/](https://www.emailjs.com/).
- Replace `"your_service_id"`, `"your_template_id"`, `"your_public_key"` in `Contact.jsx`.

## Add your Resume

Place your PDF in `/public/Jitendra_Kumar_Resume.pdf`.

## Customization

- Edit `src/data/projects.js` for your projects.
- Edit `src/data/testimonials.js` for testimonials.
- Add blog posts in `src/data/blog.js`.
